# Projeto-final-séries-temporais

Ajuste de dois modelos clássicos de séries temporais:  Holt-Winters (aditivo e multiplicativo) e SARIMA, a partir de dados mensais de consumo elétrico residencial no Brasil. O trabalho teve como objetivo diagnosticar e comparar o desempenho desses dois métodos, avaliando os pressupostos de cada modelo e a qualidade de suas previsões por métricas de erro como RMSE, MAE e MAPE.
